# Tugas Mapel Sejarah

## Deskripsi
Ini adalah proyek untuk tugas Mapel Sejarah yang dikerjakan oleh kelompok berikut:
- Zahra
- Mentari
- Dini
- Zaed

Tugas ini merupakan bagian dari pelajaran Sejarah yang dipandu oleh **Pak Singgih**.
